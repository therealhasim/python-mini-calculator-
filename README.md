# Python Calculator

This is a simple calculator built using Python.

## Features
- Addition, Subtraction, Multiplication, Division
- Continuous execution using loop
- Handles division by zero
- User-friendly input

## How to Run
1. Run the Python file
2. Enter two numbers
3. Choose operation (+, -, *, /)
